📖 Tutoriel détection Dofus

Configurez une référence fiable en quelques étapes.

Guide de configuration

Tutoriel vidéo
Ouvrir la vidéo directement
①Lancer Dofus

Lancez Dofus normalement et gardez votre fenêtre à une taille stable.

②Choisir la fenêtre

Cliquez sur Choisir la fenêtre Dofus, puis sélectionnez Dofus dans le partage.

③Capturer la référence

Affichez le dialogue PNJ final, puis cliquez sur Capturer depuis Dofus.

④Sélectionner une zone stable

Privilégiez le portrait PNJ, le nom PNJ ou un élément fixe. Évitez toute la fenêtre et les éléments animés.

⑤Tester la détection

Cliquez sur Tester la détection et vérifiez que le score est correct.

⑥Activer la détection

Activez ensuite la détection automatique. Le cooldown évite les doubles ajouts.

Conseils

✓ Gardez toujours la même taille de fenêtre Dofus

✓ Utilisez une petite zone stable

✓ Refaites la capture si la détection devient imprécise

✓ Testez avant d'activer définitivement

Sauvegarde cloud

Les membres synchronisent leur tracker en ligne. Les invités restent en sauvegarde locale navigateur.

Mode invité local Votre progression est conservée uniquement dans ce navigateur. Sans compte, elle peut être perdue si le cache est supprimé ou si vous changez d'appareil.
Sauvegarde locale active
Chaque action est sauvegardée localement. Si vous êtes connecté, elle est aussi envoyée automatiquement dans le cloud. En invité, le tracker reste local à vos risques.
